# QuantumPulse AI ⚡

**QuantumPulse AI** is a state-of-the-art web workspace featuring an expressive AI Assistant, interactive Excel/Spreadsheet Engine, multi-chart generation, AI image creation, document analysis, and cross-device cloud sync.

Created & Owned by **Saksham Sujas Shah**.

---

## 🌟 Key Features

- 💬 **Expressive Conversational AI**: Natural, warm AI responses with markdown formatting and emojis.
- 📊 **Master Executive Excel Engine**: Auto-generates 3-sheet formatted Excel workbooks (`.xlsx`) with dark slate executive banners, colorful KPI cards, status badge styling, and embedded native charts (Line, Bar, Doughnut).
- 📈 **Interactive In-Browser Charts**: Switch dynamically between 6 chart types (Bar, Line, Pie, Donut, Radar, Polar).
- 🎨 **AI Image Generation**: Built-in `/image` command for generating high-definition digital artwork.
- 📱 **Gemini-Style Wide UI & Conversations Panel**: Complete left sidebar for managing past conversations, plus a large reading view.
- ☁️ **Cloud Sync & Account Authentication**: Multi-device sync using Puter Cloud Key-Value storage and local user management.
- 📎 **File & Document Upload**: Upload CSV, Excel, PDF, text, or images for AI analysis.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/sakshamshah1631-hue/QuantumPulseAI.git

# Install dependencies
npm install

# Start development server
npm run dev
```

## 💻 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism & Radial Gradients), JavaScript (ES6 Modules)
- **Libraries**: ExcelJS, Chart.js, PapaParse, FontAwesome, Puter.js
- **Bundler**: Vite
